
# Book Review Web App (MERN Stack)

## Tech Stack
- Frontend: React.js
- Backend: Node.js, Express
- Database: MongoDB
- Auth: JWT

## Features
- User authentication (JWT)
- Role-based access: user/admin
- Book listing and management
- Book review and rating
- Responsive design

## How to Run

### Backend

cd backend
npm install
cp .env.example .env
npm run dev
